Welcome to Ana's Word Cascade Game!

How to Play:
Round 1 => 13s, Round 2 => 12s, etc.
6 Consonants + 4 Vowels per task, form words for points. Leftover letters go to the Bin.
Then Bin-Clearing Phase => 25% points each valid word from the bin. If bin overflows => Game Over!

Special Code (Testing):
A 2-letter “word” matching the first + last letter of the Active Row (in normal mode) done twice ends the round.
