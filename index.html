Welcome to Ana's Word Cascade Game!

How to Play:
Round 1 => 13 seconds, Round 2 => 12, etc.
6 Consonants + 4 Vowels each task. Words add points, leftover letters go to the Bin.
Then Bin-Clearing => 25% points per word, removing letters from the bin. If bin overflows => game over!

Cheat Code (Testing):
A 2-letter “word” matching the first & last letter of the Active Row, done twice, ends the round immediately.
